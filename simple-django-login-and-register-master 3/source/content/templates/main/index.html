Project Description↵


This neural network visualization website can help people who learn neural networks quickly understand and master the concept of neural networks and can also be used as a teaching aid tool for teachers in teaching.

User can create and save a neural network model, and others can retrieve this model through our website.

There’s also a discussion board for users to discuss the content of neural network model. Users can post their questions on the discussion board, and other users can answer these questions.
